Comunidad Y Recursos de Aprendizajes

  • PHP.net: Es el sitio oficial de PHP y ofrece una documentación muy completa sobre el lenguaje, incluyendo tutoriales, guías de referencia y manuales. También cuenta con un foro donde puedes hacer preguntas y discutir temas relacionados con PHP.
  • Stack Overflow: Esta plataforma de preguntas y respuestas es muy popular entre los desarrolladores. Hay una gran cantidad de preguntas etiquetadas como PHP, y es probable que encuentres respuestas útiles a tus dudas. Además, puedes hacer tus propias preguntas si necesitas ayuda específica.
  • PHP The Right Way: Es una guía comunitaria que ofrece mejores prácticas y consejos para escribir código PHP de manera eficiente y segura. Es una lectura recomendada para desarrolladores de todos los niveles de experiencia.
  • PHP Reddit: La comunidad de PHP en Reddit ofrece un lugar para discutir noticias, compartir recursos y hacer preguntas sobre PHP. Puedes unirte a subreddits como r/PHP para participar en la conversación.
  • PHPClasses.org: Este sitio ofrece una colección de clases y scripts PHP listos para usar, así como tutoriales y artículos escritos por la comunidad.
  • PHP Podcasts: Hay varios podcasts dedicados a PHP que pueden ser útiles para mantenerse al día con las últimas tendencias y técnicas en el desarrollo PHP. Algunos ejemplos son PHP Roundtable y Voices of the ElePHPant.
  • Cursos en línea: Plataformas como Udemy, Coursera y LinkedIn Learning ofrecen una variedad de cursos de PHP, desde principiantes hasta avanzados, impartidos por expertos en la materia.

Funciones

PHP se trata de un procesador de hipertexto que se utiliza frecuentemente para el desarrollo de sitios web de contenido dinámico. Se trata de un lenguaje de programación backend, es decir, que tiene que ver con lo que no puede ver el usuario, ya que es el encargado de conectar con una base de datos y un servidor. Las funciones de PHP, son acciones que se realizan de manera independiente. Son acciones que hacen que el código sea más legible y sencillo de depurar. Existen dos tipos de funciones: las externas y las internas, es decir, las que no se incluyen en el script y las que sí se incluyen.
